Discord Build Info JS API

A miniature and concise API for returning the build information on Discord's clients.

About

The API has three dynamic endpoints which correspond to build/{release-channel}, and return the build information accordingly.

Use

Based on the library, Discord-Build-Info-JS, this API makes it extremely easy on getting Discord's clients' build information like the build number, hash, and ID.

Mini-Documentation

To get the build information from the API, simply make a GET request on the endpoint (/build/{release-channel}) with any module or library that can perform that and the information is returned. Other requests will be rejected.
